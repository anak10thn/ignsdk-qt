TENTANG

IGN-Sudoers adalah mini aplikasi untuk memanajemen user yang akan ditambahkan atau dihilangkan sebagai pengguna biasa yang dapat menggunakan hak root sehingga dapat melakukan perintah perintah milik root.

IGOS

IGOS Nusantara mempunyai default pengguna dengan codename IGOS, IGOS dapat melakukan perintah-perintah root dengan tanpa menggunakan password.
Cara menggunakannya adalah dengan menekan button aktif pada TAB IGOS.

USER

IGOS Nusantara menyediakan manajemen pengguna yang lain, agar pengguna lain dapat melakukan perintah-perintah root dengan menggunakan password.
