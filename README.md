#IGN SDK (http://ignsdk.web.id)#
IGN (IGOS Nusantara) SDK is an app runtime. You can write native apps in HTML and Javascript with IGNSDK.

##Build##
```
$sudo yum install qt-devel qtwebkit-devel qjson
$cd ign-sdk-directory
$qmake
$make
```
